# yummba-webApp

**BACKEND**: NodeJS <br/>
**FRONTEND**: EJS <br/>
**DATABASE**: MongoDB <br/>
**PAYMENT**: Stripe <br/>
**HOSTED ON**: Heroku<br/>

### Description
An e-commerce web application for Yummba, a brand that specializes in energy bars. Users can register, add to cart, place & track orders.
